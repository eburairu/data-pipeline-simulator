# 機能仕様書: シミュレーションタブからのBIダッシュボード削除

**機能ブランチ**: `016-simulation-bi-removal`
**作成日**: 2026-02-07
**ステータス**: 完了 (Completed)
**入力**: ユーザー要望

## 背景

シミュレーション制御を行う「シミュレーション」タブ内にBIダッシュボードが表示されていたが、画面が複雑になるため、これらを分離して各タブの役割を明確にする。

## 変更内容

1. **SimulationManagerのレイアウト修正**:
    - `App.tsx` 内の `SimulationManager` コンポーネントから、2カラムレイアウト（`lg:grid-cols-2`）を削除。
    - シミュレーション制御、ストレージ表示、データベース表示を全幅のシングルカラムで表示するように変更。

2. **BIダッシュボード表示の削除**:
    - `SimulationManager` 内での `BiDashboard` コンポーネントの呼び出しを削除。
    - BIダッシュボードの確認は、専用の「BIダッシュボード」タブでのみ行うように制限。

## ユーザーシナリオとテスト

### ユーザーストーリー 1 - シミュレーション制御の簡素化

ユーザーとして、シミュレーションの実行状態を広々と確認したい。

**受け入れシナリオ**:

1. **操作** 「シミュレーション」タブを選択する
2. **期待結果** シミュレーション制御パネル、ストレージビュー、データベースビューが全幅で表示される
3. **期待結果** 画面下部や右側にBIダッシュボード（グラフ等）が表示されない
4. **操作** 「BIダッシュボード」タブを選択する
5. **期待結果** BIダッシュボードが表示される
